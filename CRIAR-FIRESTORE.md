# Como criar o Banco de Dados Firestore no Firebase

Siga estes passos para ativar o Firestore:

## Passo 1: Acesse o Console do Firebase

1. Vá para https://console.firebase.google.com/
2. Clique no seu projeto: **hortifruti-pdv**

## Passo 2: Vá para o Firestore Database

No menu lateral esquerdo, procure e clique em:
**"Firestore Database"** (pode estar como "Cloud Firestore" em inglês)

## Passo 3: Criar o Banco de Dados

1. Clique no botão **"Criar banco de dados"** (Create database)
2. Uma janela vai abrir perguntando sobre localização
3. Selecione: **nam5 (United States)** - ou fica com o padrão
4. Clique em **"Próximo"** (Next)

## Passo 4: Configure as Regras de Segurança

1. Selecione a opção: **"Iniciar no modo de teste"** (Start in test mode)
   - Isso permite leitura e escrita sem bloqueios
2. Clique em **"Criar"** (Create)

## Passo 5: Aguarde a Criação

O Firebase vai criar o banco de dados. Pode levar alguns segundos.
